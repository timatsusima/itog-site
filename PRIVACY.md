# Политика конфиденциальности · Itog

_Последнее обновление: 7 мая 2026_

Itog — приложение для macOS, которое локально записывает совещания и
формирует протокол. Этот документ описывает, какие данные мы обрабатываем
и куда они уходят.

## Кратко (TL;DR)

- **Аудио и транскрипт** никогда не покидают ваш Mac. Запись хранится в
  `~/Library/Application Support/Itog/Recordings/`. Транскрипция запускается
  локально через [WhisperKit](https://github.com/argmaxinc/WhisperKit) на
  Apple Neural Engine.
- **Текст протокола** (краткое резюме + action items) формируется выбранным
  вами провайдером: Anthropic, OpenAI или локальная Ollama. В случае
  облачных провайдеров туда уходит **только текст транскрипта**, не аудио.
- Ключи API хранятся локально в `UserDefaults` вашего пользователя.
- Itog не использует аналитику, рекламные SDK и не отправляет ваши данные
  третьим сторонам сверх того, что вы явно настроили (Notion, Telegram,
  iCloud, RevenueCat).

## Какие данные мы собираем

Itog **не собирает** персональные данные на стороне разработчика.

В рантайме приложение работает со следующими локальными данными:

- Аудиозаписи встреч (WAV).
- Транскрипты, сегментированные по говорящим (`[Я]` / `[Собеседник]`).
- Сгенерированные протоколы в Markdown.
- Метаданные встреч: дата, длительность.
- Ваши настройки (выбор провайдера, шаблон протокола, флаги).

Все эти данные хранятся **только на вашем Mac**, в директории
`~/Library/Application Support/Itog/` и в `UserDefaults` приложения.

## Куда уходят данные

### Локально (по умолчанию)

- **Транскрипция**: WhisperKit, на устройстве. Никаких сетевых запросов.
- **Хранилище встреч**: `UserDefaults` + WAV в Application Support.

### Опционально, по вашему выбору

| Сервис | Что отправляется | Когда |
|---|---|---|
| Anthropic API | Текст транскрипта | Если выбран Claude как провайдер |
| OpenAI API | Текст транскрипта | Если выбран OpenAI как провайдер |
| Ollama (localhost) | Текст транскрипта | Если выбран Ollama (запросы не покидают Mac) |
| Notion API | Заголовок + блоки протокола | По вашему явному клику «Экспорт → Notion» |
| Telegram Bot API | Заголовок + протокол | По вашему явному клику «Экспорт → Telegram» |
| Apple iCloud (CloudKit) | Дата, длительность, транскрипт, протокол | Если включён тоггл «Синхронизация через iCloud» |
| RevenueCat | Анонимный пользовательский ID, статус подписки | Только если SDK сконфигурирован реальным API-ключом |

**Аудиофайлы (WAV) не уходят никуда**: ни в облако, ни в iCloud, ни в Notion.
Только локально.

## Разрешения macOS

Itog запрашивает следующие разрешения через стандартный TCC-механизм
macOS:

- **Микрофон** — для захвата вашего голоса.
- **Запись экрана и системного звука** — для захвата голоса собеседника
  через ScreenCaptureKit (видео-кадры мы получаем, но сразу отбрасываем —
  записываем только аудио).
- **Уведомления** — для нотификаций «Протокол готов» и «Itog обнаружил
  созвон».

## Хранение и удаление

Чтобы полностью удалить ваши данные:

1. Откройте `~/Library/Application Support/Itog/` и удалите всё.
2. Удалите `~/Library/Preferences/com.tima.itog.plist`.
3. Удалите `~/Documents/huggingface/models/argmaxinc/whisperkit-coreml/`
   (кэш моделей Whisper).
4. Если включали iCloud-синхронизацию — откройте Системные настройки →
   Apple ID → iCloud → Manage → Itog и удалите данные.
5. Удалите само приложение из `/Applications`.
